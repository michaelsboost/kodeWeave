Hello! My name is Michael Schwartz and I created kodeWeave because I wanted to have a fast and reliable application I can code with on the go. In order for that to happen it the following requirements where built in.

  1. Auto Update: If enabled all code changes made to your editor or even in the settings updates your preview in realtime.
  2. Integrated Console: kodeWeave is built with an integrated developer console that way if any errors occur you can see those errors instantly and what line that error was triggered in your code. The console can also be toggled in settings so you have the option whether you want to use it or not.
  3. Cross Platform Development: kodeWeave itself is a PWA (Progressive Web App) therefore you can easily use kodeWeave on any device from a single codebase. In addition I wanted to make sure that every application I create with kodeWeave is easily exportable as a PWA which makes cross platform development faster and easier for everyone.

    In addition kodeWeave allows you to zoom and pan the preview so you can see how your app will look or behave for a desktop even though you're developing on a phone whether you're connected to the internet or not.
  4. Emmet makes it possible to write a lot of html extremely quickly just by using abbreviations. To expand the appreviation and insert all the code all you need to do is hit the enter get on your keyboard.
  5. Pico.css and/or Tailwind CSS: Design and style your website or app extremely fast by using Pico.css and/or Tailwind CSS for your CSS style system. If you're not familiar with either Pico.css is a minimal css framework for semantic html. Meaning all the styling is already done for you (with both a light and dark theme). Tailwind CSS is a huge css library filed with utility classes for everything you can think of from display, color, box-shadow, you name it. This way if you want complete creative freedom to style your website and to do it fast Tailwind CSS is ideal especially for those coding on a phone.
  6. Share on Codepen: Codepen is a popular social development environment for front-end designers and developers.
  7. kodeWeave also comes with linting, ES6 Support, close tags, close brackets, match tags, match brackets, among many other handy goodies.
